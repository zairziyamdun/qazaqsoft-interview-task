## Что было реализовано

В ходе выполнения тестового задания был реализован полноценный модуль прохождения теста на чистом JavaScript без использования фреймворков.

Реализовано:

- загрузка вопросов из `questions.json`
- отображение прогресса прохождения теста
- навигация между вопросами
- выбор и сохранение ответов
- таймер обратного отсчёта
- автоматическое завершение теста при окончании времени
- подсчёт результата и процента прохождения
- просмотр правильных и неправильных ответов после завершения
- сохранение состояния теста через `localStorage`
- восстановление прогресса после перезагрузки страницы
- возможность полного сброса и повторного прохождения теста

## С какими проблемами столкнулся

Основные проблемы:
- корректное восстановление состояния после перезагрузки страницы;
- синхронизация таймера и завершения теста;
- сохранение выбранных ответов при возврате назад;
- корректный flow после завершения теста и перехода в режим просмотра ответов;
- разделение UI логики между экраном результата и review mode.

Дополнительно пришлось доработать структуру шаблонного проекта:
- реализовать недостающую логику `QuizEngine`;
- убрать заглушки и шаблонные `TODO`;
- привести код к более читаемому и поддерживаемому состоянию.

## Что можно было бы улучшить

Если развивать проект дальше, я бы:

- Изменения дизайна, добавления страницы начать тест
- вынес логику в отдельные модули;
- добавил обработку ошибок загрузки JSON;
- реализовал unit-тесты для `QuizEngine`;
- добавил более подробную аналитику прохождения теста.
